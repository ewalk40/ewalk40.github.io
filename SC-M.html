Simple Counter - Count Keeper - Marketing


Description

Introducing...Simple Counter - Count Keeper!

If you're like me, you're always needing to keep track of how much of something there is, for example how many Hershey Kisses you've eaten that do so you don't go overboard. Well this app has got you covered so you never have to remember a count ever again!

Features:
• Super simple interface
• Easy to use swipe gestures for increasing or decreasing count
• Large text so you never have to squint to see what number you're at

Some Examples of Things to Count:
• Number of singles during a baseball game
• How many runners at a 5K
• Cookies made for a bake sale
• Score in a cornhole game
• Favorite pieces of candy eaten

The list goes on and on!

My brother was the insperation for this app, he needed to count how many runners ran through the starting line at the start of a local 5K and it was hard for him to keep track as everyone was barreling through. So for this latest 5K, I had him use a beta version of the app and he said, "There is no way I will never have this app on my phone again, there are so many uses for it!"

My idea with this app was to make it dead simple to use. I didn't want there to be any confusion on how it worked. You click the + button when you load up the app to create a new counter, give it a name, then either swipe up or down in the next screen to increase or decrease your count, it's that simple!

Features to Come:

I hope everyone can find their own use for this app!

Enjoy!

- Eric Walker, Head Developer
